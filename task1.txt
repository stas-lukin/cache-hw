Проблемы, которые решает кеширование:

1. Высокая нагрузка на основную базу данных - кеш обрабатывает повторяющиеся запросы

2. Медленное время ответа - данные из RAM отдаются в миллисекунды

3. Дорогие вычисления - результат сложных операций сохраняется

4. Пиковые нагрузки - кеш помогает выдержать всплески трафика

5. Задержки внешних API - ответы сохраняются локально

6. Избыточное потребление CPU - тяжёлые запросы выполняются реже

7. Сетевой трафик - CDN хранит контент ближе к пользователю
